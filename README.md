# ChatGPT 生成器小程序


**⚡** **已接入最新gpt-3.5-turbo模型**

码云链接：``https://gitee.com/JN910-14/ChatGPT-miniapp``

## 🔧 环境要求

1.科学上网

2.注册获取key

## ⚙已有功能

* 日报周报生成
* 文章框架生成
* 文案标题生成
* 检讨书生成
* 代码帮写/查bug
* 方案助手
* 理由推脱
* 评价生成
* 个人总结
* 好物描述
* 肯德基V50
* ......

## 🔧项目部署

1.clone本项目（miniapp）

2.全局搜索替换域名

3.修改appid

4.上传到公众平台

5.部署服务端（见后台项目）
